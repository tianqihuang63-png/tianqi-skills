# Output Template

Use this only after the user is stable enough.
Do not rush to output structure while the user is still flooded.

## Compact closing format

### 1. 当前情绪模式

- 例如：比较后的焦虑
- 例如：受挫后的自卑
- 例如：慌张中的自我否定

### 2. 当前解释模式

- 例如：把暂时没做到解释成自己不行
- 例如：把差距解释成价值低
- 例如：把落后解释成被淘汰

### 3. 背后的真实意图

- 例如：你是真的想做出结果
- 例如：你想抓住机会，不想白白错过
- 例如：你想活得更有掌控感

### 4. 现在最值得处理的现实问题

- 例如：你现在缺的不是价值，而是阶段认知
- 例如：你现在最卡的是还没有最小验证
- 例如：你现在需要面对的不是情绪本身，而是下一步路径不清

### 5. 一个最小行动

Action must be:

- concrete
- small
- immediate
- reality-based

Bad:

- "重建自信"
- "以后更努力"

Good:

- "今天把你的副业当前卡点列成 3 条。"
- "这周找 1 个已经做过的人问清楚第一步验证怎么做。"
- "今天只定义出你的第一个最小可发布版本。"

### 6. 支持语

Support must be grounded, not generic.

Good examples:

- "你现在不是不行，你是先被结果刺痛了，还没把问题看清。"
- "你会这么难受，说明你真的在乎；你不需要今天赢，你需要今天往前。"
